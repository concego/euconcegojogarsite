\# Euconcegojogar



\[cite\_start]\*\*Acessibilidade e inclusão: de gamers para gamers.\*\* \[cite: 298]



\[cite\_start]Este é o repositório oficial do site do projeto Euconcegojogar, uma iniciativa criada por pessoas cegas para promover acessibilidade digital em jogos, aplicativos e plataformas interativas\[cite: 299].



\## Missão

\[cite\_start]Garantir que pessoas cegas tenham acesso pleno e digno às experiências digitais, com autonomia, respeito e inclusão\[cite: 313].
